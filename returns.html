Returns & Refund Policy

Our return and refund guidelines

Return Window: 30 days from delivery date
Refund Timeline: 5-7 business days after return approved

Eligible Items for Return

  • Unused items with original tags attached
  • Items in original packaging
  • Wrong size received (provided you ordered correctly)
  • Defective or damaged products
  • Wrong product delivered

Items Not Eligible for Return

  • Used or washed items
  • Items without original tags
  • Innerwear, socks, and accessories
  • Items damaged due to customer misuse
  • Customized or altered products
  • Sale items (unless defective)

How to Request a Return

  • Contact us via email or phone with your Order ID
  • Provide photos of the item (if defective)
  • Wait for return approval confirmation (within 24-48 hours)
  • Pack the item securely with all tags
  • Ship the item back using our provided label or courier

Refund Process

  • Once we receive and inspect your return (3-5 business days)
  • Refund approved → Amount credited within 5-7 business days
  • Refund via original payment method
  • You will receive email/SMS confirmation at each step

Exchange Policy

We offer size exchanges for the same product. Simply return the current size and place a new order for the desired size. Color exchanges are not available.

Return Shipping

  • Defective/Wrong Product: Free return shipping (we'll arrange pickup)
  • Size Change/Other: Customer bears shipping cost (₹99 deduction)

Refund Options

  • Original Payment Method: Credit/Debit card, UPI, Net Banking
  • Store Credit: 10% extra credit for future purchase (optional)
